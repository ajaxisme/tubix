# tubix
An app to show youtube video and lyrics at the same time

## Technologies
1. React - front end
2. Flask - API backend

## How to run
__API__
```
1. cd python-lyrics/
2. virtualenv lyrics
3. source ./lyrics/bin/activate
4. pip install bs4 requests Flask flask-cors
5. python server.py
```

__Web App__
```
1. npm install
2. yarn start or npm start

App running on localhost:8080
```
